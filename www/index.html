BTrace

BTrace is a dynamic tracing system for the Java platform. BTrace lets user to specify tracing of a running Java program (similar to DTrace for OpenSolaris applications and OS) and dynamically instruments target application classes to inject tracing code. Tracing code is expressed in Java programming language. There is also integration with the DTrace system for the OpenSolaris platform. There will be a VisualVM plugin for BTrace back-end - so that the users can trace their application from the VisualVM client.
