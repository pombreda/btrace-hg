NetBeans BTrace Project Setup

This document describes how to setup the BTrace NetBeans project

  1. Checkout the sources from CVS
    2. Open the checked-out sources in NetBeans
  2. Bring up the project properties and select Java Sources Classpath - you should see this picture
  3. Change the ${tools.jar} entry to point to the location of tools.jar in your environment
  4. Confirm the changes you have made and you can start working with the BTrace project
